# Canteen Automation System

## 📌 Overview
A web-based system developed to automate food ordering and canteen management processes.

## 🚀 Features
- Online food ordering system
- Menu management
- Customer order tracking
- Admin panel for managing items

## 🧠 Key Highlights
- Reduced manual errors in order processing
- Improved efficiency of canteen operations
- Real-time order handling

## 🛠️ Technologies Used
- HTML
- CSS
- PHP
- MySQL
